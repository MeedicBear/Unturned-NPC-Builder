## 📦 Overview

**Unturned NPC Builder** is a desktop application built with **Electron** that allows players, server owners, and modders to create fully‑functional NPCs without manually writing `.dat` files.

The app includes:

- NPC Overview Editor  
- Dialogue Builder  
- Vendor Buy/Sell Editor  
- Flag Manager  
- Live Preview of all `.dat` files  
- One‑click Export to NPC folder + ZIP  

This tool removes the complexity of Unturned’s NPC system and replaces it with a clean, visual interface.

---

## ✨ Features

### 🧍 NPC Editor
- NPC ID, Name, Dialogue ID  
- Auto‑formatted `.dat` output  

### 💬 Dialogue Builder
- Message text  
- Unlimited responses  
- Optional next‑dialogue linking  
- Auto‑generated `Dialogue.dat`

### 🛒 Vendor Builder
- Buy items  
- Sell items  
- Price editor  
- Auto‑formatted `Vendor.dat`

### 🚩 Flag Manager
- Add/remove flags  
- Auto‑formatted `Flags.dat`

### 📤 Export System
- Generates:
  - `NPC.dat`
  - `Dialogue.dat`
  - `Vendor.dat`
  - `Flags.dat`
- Creates a folder + ZIP archive  
- Ready to drop into your Unturned server or mod project
